Cheatsheet

Cheatsheet van map naar Github

1. Eerst openen we de command line (CMD).

2. En dan type je of cd.. of cd en de naam van de wat waar je het in hebt gedaan.

3. Git init= met git init maak je een nieuwe repository.

4. Git status= met git status zie je wat er in de repository.

5. Git add= met git add plaats je een bestand in de repository en als je (git add .) doet dan sla je alles op in de map.

6. Git commit= met git commit zet je de document vast.

7. Git log= met git log zie je de commits.

8. Git push= met git push zet je je laatste commant in github.

9.Volg dan de stappen die in de repositorie staan in GitHub, Typ deze dan in je cmd. (git remote add origin https://github.com/HalilDogan7007/les07.git) Git branch -m main = git branch vul je een woord in die verwijst wat je hebt gedaan of wat je hebt gemaakt. Git push -u origin main de u betekend de upstream en met de git push houd je de gegevens up-to-date.

10. Als het goed is heb je nu je nieuwe repositorie in GitHub aan gemaakt.
